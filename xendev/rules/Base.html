Timer

Using LiveSplit is required for getting the time without loads, adding 1 tick to each saveload & automatic timer. Latest SourceSplit (3.3.3+) must be used. The SourceSplit settings must be set according to this guide.

Misc.

Using scripts, macros, turbo buttons, hyper-scrolling and so on is not allowed.

Using two types of controllers at the same time is not allowed. This means you cannot use keyboard and mouse at the same time with a controller (except in HL2 Category Extensions)
