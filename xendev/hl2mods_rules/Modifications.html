Textures

Any texture can be modified, as long as it is not used to create easier line-ups or help visibility of certain areas.

Sounds

Any sound can be modified, as long as it does not make sound cues easier, such as making certain sounds or voice lines louder to help timing tricks.

HUD

You are allowed to change HUD's color and font.
You are allowed to remove the white flashes caused by taking damage on toxic goo.
Disabling red flashes from taking damage is not allowed.
Enlargement of the cl_showpos text with the sourcescheme.res text file is allowed.

Models

You are not allowed to change any NPC Models.
You are allowed to change viewmodels (weapons).

Files

You are allowed to replace FileSystem_Stdio.dll in the bin folder to prevent save stutter.
Using maps from different versions is not allowed, this means you must use maps consistent with the version of the game you are running.
